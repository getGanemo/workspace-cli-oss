# Code of Conduct

This project has adopted the **Contributor Covenant, version 2.1**.

The full text is maintained at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/> and is incorporated here by reference. We follow it verbatim — please read it there as the canonical version.

## Reporting

Reports of conduct violations should go to **fernando@ganemo.co**. Reports are handled confidentially. We aim to acknowledge within 3 business days and resolve within 30 days.

## Scope

The Code of Conduct applies in all project spaces (issues, PRs, discussions, the docs site, any chat or video channel run by the project) and any time someone is representing the project in public.
